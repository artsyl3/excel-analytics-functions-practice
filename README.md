# Excel Data Analysis Practice

This project contains a collection of Excel exercises focused on learning and practicing important Excel functions commonly used in data analysis.

The dataset simulates employee information including names, job roles, salary data, and employment dates. Using this dataset, several Excel functions were applied to analyze and manipulate the data.

## Dataset Overview

The dataset includes the following columns:

- EmployeeID
- First Name
- Last Name
- Age
- Gender
- Job Title
- Salary
- Start Date
- End Date
- Email

This dataset was used to practice multiple Excel formulas and data manipulation techniques.

---

# Excel Concepts Practiced

## 1. SUBSTITUTE

Used to replace specific text within a string.

Example:
Replace parts of email addresses or modify text values inside cells.

Skills practiced:
- Text cleaning
- Replacing characters in strings

---

## 2. SUM & SUMIF

Used to calculate totals and conditional sums.

Examples:
- Total salary calculations
- Summing values based on conditions

Skills practiced:
- Aggregating numeric data
- Conditional calculations

---

## 3. COUNT & COUNTIF

Used to count values and count based on conditions.

Examples:
- Counting number of employees
- Counting employees by gender

Skills practiced:
- Data summarization
- Conditional counting

---

## 4. CONCATENATE

Used to combine multiple text values into one cell.

Example:
Combining first name and last name to create a full name.

Skills practiced:
- String manipulation
- Data formatting

---

## 5. LEFT & RIGHT Functions

Used to extract characters from the beginning or end of a text string.

Example:
Extracting the first or last characters from names.

Skills practiced:
- Text extraction
- String parsing

---

## 6. MAX & MIN

Used to find the highest and lowest values in a dataset.

Example:
Finding the maximum and minimum salary values.

Skills practiced:
- Data comparison
- Identifying extremes in datasets

---

## 7. DAYS & NETWORKDAYS

Used to calculate the difference between dates.

Examples:
- Calculating employment duration
- Counting working days between two dates

Skills practiced:
- Date calculations
- Business day analysis
